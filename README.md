# Рейтинг державних сервісів України, калькулятор витрат, формування пакетів

Навіяно обговоренням в Discord чяті ДемОрди

Українська держава надає громадянам тисячі сервісів, значна кількість яких громадянами навіть не викоростувається (держконтрольалмазопостач та інші), частина є дорожчими по факту за приватні (собівартість розкритя злочину нацполіцією 100000 грн за штуку, за реформою супрун терапевт коштує 370 грн за місяць на одну особу тощо). Щоб привести це до ладу можна зробити громадьский сервіс з наступним функціоналом

## Перелік всіх сервісів що надає держава, з вартістью бюджетних витрат на кожний

## Рейтингове голосування за сервіси
Як в громраду НАБУ, кожен голосує за 50 (або за необмежену кількість) потрібних йому від держави сервісів, ті що набрали менше визначенного порогу, йдуть на виліт. Також сайт автоматично рахує наскільки зменшуються витрати бюджету якщо позбутися непотрібних сервісів

## Формування податкових пакетів
Кожен користувач може обрати перелік послуг який він хоче отримувати від держави, сайт автоматично розраховує скільки громадянин має сплатити за це податків
